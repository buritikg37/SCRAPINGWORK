REDAGRO SCRAPING WEB

Contenido:
- RedAgroScrapingWeb.exe
- redagro.ico
- GUIA_EJECUCION.md

Uso:
1. Haz doble clic en RedAgroScrapingWeb.exe
2. La aplicacion abrira el navegador en http://127.0.0.1:5000
3. Desde la web puedes buscar categorias, extraer resultados y exportar a Excel

Notas:
- Requiere Google Chrome instalado
- Los archivos JSON y CSV se guardan junto al ejecutable
- Cada categoria buscada se guarda tambien en la carpeta busquedas
